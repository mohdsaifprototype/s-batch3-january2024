List attributes

NOTE: Attributes are always mention inside the opening tag!

type attributes values for <ul>

Syntax:
<ul type="values"> ... </ul>

Note: Underline means default value of the attribute
  • disc
  • circle
  • square
  • none

type attributes values for <ol>

Syntax:
<ol type="values"> ... </ol>

Note: Underline means default value of the attribute
  1. 1
  2. i
  3. I
  4. a
  5. A

start attribute for ol

Syntax:
<ol type="1" start="11">

  1. a big red truck
  2. an aeroplane that flies
  3. a nice soft toy
  4. a drum set
  5. a Walkman
  6. extra pocket money
